SNCF API

Les données sur les commerces de proximité sont obtenues à partir d'une recherche par code postal de l'API publique RATP. Ces commerces de proximité (librairies, points presse, débitants de tabac...) sont à la disposition des voyageurs pour l'achat de carnets de Tickets t+ et les rechargements Navigo Mois et Semaine ainsi que des passes Navigo Découverte.

Les données de la page de Contact sont extraites de l'API Github.

Framework JS: Angular 17
Framework CSS: Tailwind
Le Testing: Jasmine, Karma
